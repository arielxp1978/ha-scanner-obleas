# ha-scanner-obleas

Repositorio de add-ons para Home Assistant — Scanner de Obleas GNC.

## Cómo agregar este repositorio en Home Assistant

1. Settings → Add-ons → Add-on Store → ⋮ → Repositories
2. Pegar: `https://github.com/arielxp/ha-scanner-obleas`
3. Buscar "Scanner Obleas GNC" e instalar

## Add-ons disponibles

### Scanner Obleas GNC
Escanea obleas GNC de ENARGAS desde tu Raspberry Pi con Home Assistant.
